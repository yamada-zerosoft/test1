# シンプル格闘ゲーム

HTML5 Canvas とプレーン JavaScript で動く2人用のミニ格闘ゲームです。ローカルでページを開くだけで遊べます。

## 遊び方
- Player1: A / D で移動、W でジャンプ、J で攻撃、K で防御
- Player2: ← / → で移動、↑ でジャンプ、1 で攻撃、2 で防御
- スタミナが少ないと攻撃力が落ちます。防御中はダメージが軽減されます。
- 体力が0になるとそのラウンドは負け。リセットボタンで次のラウンドを開始できます。

## 動かし方
```bash
python -m http.server 8000
# ブラウザで http://localhost:8000 にアクセス
```
